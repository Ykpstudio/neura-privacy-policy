#  Privacy Policy for Neura (AI Coding & Startup Assistant)

**Last Updated:** November 6, 2025

Welcome to **Neura**, an intelligent AI assistant designed to support software development, AI, and creative startup projects.  
This Privacy Policy explains how we handle information and ensure user data safety.

---

## 1. Information We Collect
- **User Inputs:** Text or code you provide while asking questions or requesting help.  
- **Non-personal Data:** Usage analytics and performance metrics for debugging and improvement.  
- **Optional Data:** Any files or project details you voluntarily share (not automatically collected).  

We **do not** store or access personal identifiers such as names, emails, or payment information.

---

## 2. How We Use the Data
- Generate responses, explanations, or code suggestions

No data is sold, shared, or reused for marketing or profiling.

---

## 3. Data Storage & Security

- No long-term records are kept outside of OpenAI’s secure systems.
